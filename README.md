# estudos-notebooklm
Projeto Dio utilizando notebookLM

Nesse caderno tematico vamos falar sobre o feminicidio.
O objetivo desse estudo é falar sobre esse assunto tão delicado, quais sao as providencias que o governo esta fazendo para auxiliar e como procurar ajuda caso seja necessario.
Lembrar tambem que o feminicidio nem sempre se inicia com agressoes fisicas, na maioria das vezes se inicia com ações verbais e psicologicas.
